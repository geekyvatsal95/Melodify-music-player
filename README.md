# 🎵 Melodify Music Player

A modern and responsive music player UI built using HTML & CSS.

## 🚀 Features

* Clean and minimal UI
* Sidebar navigation
* Music cards layout
* Sticky navigation bar
* Interactive music player section
* Responsive design

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Layout)
